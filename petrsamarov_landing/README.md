# petrsamarov.com — v2.0

Static bilingual author website for Petr Samarov.

## Structure

- `/index.html` — Russian version
- `/en/index.html` — English version
- `/assets/` — images, favicon, CSS
- `/sitemap.xml` — sitemap with RU and EN pages
- `/robots.txt` — crawler rules
- `/404.html` — custom 404 page

## Deployment

Push the contents of this folder to the GitHub repository connected to Vercel. Vercel will redeploy automatically.

If the repository keeps files in the `petrsamarov_landing` subfolder, replace the contents of that subfolder with these files.
